# Universitet Murojaat Tizimi

Universitet Murojaat Tizimi — bu talabalar va fakultet xodimlari o‘rtasidagi murojaat jarayonini avtomatlashtiruvchi web ilova.

## Loyiha Maqsadi

Talabalar o‘z muammolari, savollari yoki rasmiy arizalarini onlayn tarzda yuborishi va fakultet xodimlari tomonidan ko‘rib chiqilishi uchun yagona tizim yaratish.

## Asosiy Imkoniyatlar

- Talabalar ro‘yxatdan o‘tishi
- Tizimga login qilish
- Fakultet asosida murojaat yuborish
- Real vaqt chat orqali muloqot
- Murojaat statusini kuzatish
- Xodim tomonidan "Masala hal etildi" funksiyasi
- Talaba tasdiqlashi orqali murojaatni yopish
- Yopilgan murojaatlarni arxivlash

## Foydalanuvchi Rollari

### Student
- Murojaat yuboradi
- Javoblarni ko‘radi
- Masala hal etilganini tasdiqlaydi

### Staff
- O‘z fakultetiga tegishli murojaatlarni ko‘radi
- Chat orqali javob beradi
- Masalani hal etilgan deb belgilaydi

### Admin
- Tizimni boshqaradi
- Statistika va nazoratni amalga oshiradi

## Murojaat Jarayoni

1. Talaba murojaat yuboradi
2. Fakultet xodimiga bildirishnoma boradi
3. Chat orqali yozishma boshlanadi
4. Xodim masalani hal etilgan deb belgilaydi
5. Talaba tasdiqlaydi
6. Murojaat yopiladi va arxivga o‘tadi

## Texnologiyalar

Frontend: React / Next.js  
Backend: Node.js (Express yoki NestJS)  
Ma’lumotlar bazasi: PostgreSQL  
Autentifikatsiya: JWT  
Real vaqt chat: WebSocket (Socket.io)

## Kelajakdagi Rejalar

- Email xabarnomalar
- Fayl yuklash imkoniyati
- Statistik dashboard
- Baholash tizimi
- Mobil versiya
